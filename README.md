Session-based authentication with login/logout flow
Food Preparation module — logs date, food name, quantity, and auto-detects day type
Consumption module — records intake across 4 time slots, auto-computes leftover and classifies waste type
Rule-based recommendation engine for waste handling (reuse, donation, animal feed, composting, bio-gas)
Dashboard with real-time KPIs: total prepared/consumed/waste, waste %, today & monthly waste, efficiency score, animated counters
Forecasting module predicting next-day preparation quantity from historical consumption + day-type buffer (10%/20%)
Reports module with daily/weekly/monthly/time-wise/date-range waste analytics, visualized via Chart.js bar charts
Responsive glassmorphism UI with collapsible sidebar
Relational schema with foreign key constraints across users, food_preparation, consumption, day_type tables

live demo : foodwastemanagementsystem.page.gd
